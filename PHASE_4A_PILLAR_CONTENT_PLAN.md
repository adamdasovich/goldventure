## What is Pillar Content?

Pillar content is a comprehensive, authoritative piece of long-form content that:
- Covers a topic in-depth (3,000-5,000 words)
- Targets high-volume, high-intent keywords
- Serves as a foundational resource that other content links to
- Establishes domain authority and expertise
- Generates backlinks naturally due to comprehensive coverage

## Writer Guidelines

### **Tone and Style**:
- Professional but accessible
- Educational, not promotional
- Data-driven with statistics and examples
- Avoid jargon without explanation
- Use active voice
- Short paragraphs (2-4 sentences)

### **Research Standards**:
- Cite all statistics with sources
- Use primary sources (SEDAR+, TSX, government data)
- Verify facts with multiple sources
- Update data to 2026
- Link to authoritative external sources

### **Formatting**:
- Use descriptive headers (not "Introduction" but "What Are Junior Gold Mining Companies?")
- Include numbered and bulleted lists
- Add tables for comparisons
- Use bold for emphasis (sparingly)
- Include pull quotes for key insights
